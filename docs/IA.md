# IA: 정보 구조 정의

## 1. 메뉴 구조 (Sitemap)
- **메인 입력화면 (Survey Form)**
    - Phase 1: 기관 기본 정보 (시군, 기관명, 담당자)
    - Phase 2: 운영 현황 (종사자, 대상자, 특화, 단기집중)
    - Phase 3: 배정 인원 (기준 TO, 변경 사항)
    - Phase 4: 최종 검토 (종합 요약, 제출 동의)
- **지도 시각화 대시보드 (Map Dashboard - Standalone Project)**
    - 통계 요약 섹션
    - 상세 지표 카드
    - 지도 시각화 엔진 (테마/모드 전환)
    - 이상 데이터 탐지 및 미제출 목록

## 2. 데이터 구조 (Data Schema)
- **기관 마스터**: 시군, 기관명, 기관코드, 거점여부.
- **월별 제출 데이터**: 성별/유형별 종사자 및 대상자 수, 단기집중 단계별 현황.
- **배정 인력 데이터**: 정부 배정 인원, 변경 이력.
- **지도 데이터**: 경남 18개 시군 공간 정보 (SVG Path), 시군별 통계 매핑 데이터.
