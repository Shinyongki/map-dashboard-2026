# Use Case: 노인맞춤돌봄서비스 통합 시스템

## 1. 사용자 시나리오: 수행기관 담당자
1. **로그인**: 서비스 URL 접속 (별도 가입 없이 시군 선택으로 시작).
2. **현황 입력**: 종사자(남/여), 대상자(일반/중점/특화) 인원 입력.
3. **지능형 Q&A 활용**:
    - **공문 조회**: 최근 발송된 공문 목록과 AI가 요약한 핵심 내용 확인.
    - **질문하기**: 지침이나 공문 내용에 대해 자유롭게 질문.
    - **답변 확인**: AI가 작성한 답변 초안과 관련 공문 근거 조항을 즉시 확인.

## 2. 사용자 시나리오: 관리자 (광역지원기관)
1. **대시보드 접속**: 독립된 지도 대시보드 URL (`/map`)로 직접 접속하여 인증 후 진입.
2. **공문 및 지식 관리**:
    - **공문 업로드**: PDF 파일을 업로드하면 AI가 자동으로 텍스트를 추출하고 요약 생성.
    - **FAQ 승인**: 빈번한 질문에 대해 AI가 제안한 FAQ를 검토 후 전체 공개로 전환.
3. **AI 보고서 및 브리핑**:
    - 제출된 현황 데이터를 기반으로 AI가 자동 생성한 월간 분석 브리핑 확인.
4. **지도 기반 공간 분석**: 
    - 테마 전환(Infographic 모드 등)을 통한 고품질 시각화 조회.
    - **기상/재난 모니터링**: 폭염 특보 발령 시, 해당 지역 독거노인 안전 확인 우선순위 책정.
