# Use Case: 노인맞춤돌봄서비스 통합 시스템

## 1. 사용자 시나리오: 수행기관 담당자
1. **로그인**: 서비스 URL 접속 (별도 가입 없이 시군 선택으로 시작).
2. **현황 입력**: 종사자(남/여), 대상자(일반/중점/특화) 인원 입력.
3. **거점 업무 보고**: 단기집중서비스 이용자 현황(신규, 만료, 포기) 보고.
4. **배정 변경 보고**: 정부 배정 TO 변경 시 증빙과 함께 보고.
5. **검토 및 제출**: 입력 내역 요약 확인 후 최종 제출.

## 2. 사용자 시나리오: 관리자 (광역지원기관)
1. **대시보드 접속**: 독립된 지도 대시보드 URL (`/map`)로 직접 접속하여 인증 후 진입.
2. **월별 전체 분석**: 전체 제출률 및 인력/기능별 통계 확인.
3. **지도 기반 공간 분석**: 
    - 테마 전환(Infographic 모드 등)을 통한 고품질 시각화 조회.
    - 지역별 인력 부족 또는 이용자 쏠림 현상(수급 균형) 파악.
    - **[NEW] 기상/재난 모니터링**: 폭염 특보 발령 시, 해당 지역 독거노인 안전 확인 우선순위 책정.
    - **[NEW] 자원 연계**: 필요 시 관내 노인의료복지시설 목록을 즉시 조회하여 연계.
4. **데이터 품질 관리**: 전월 대비 급격한 변동 지역 확인 및 미제출 기관 독려.
5. **관리 및 보고**: 
    - 대시보드에서 **관리자 페이지**로 즉시 이동하여 데이터 수정.
    - 월간 결과 취합 보고를 위한 엑셀/이미지 다운로드.
