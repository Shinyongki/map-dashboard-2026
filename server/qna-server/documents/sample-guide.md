# 2026년도 긴급복지지원사업 안내

## 1. 사업 개요
위기상황에 처한 사람에게 일시적으로 신속하게 지원하여 생계유지 등을 돕는 사업입니다.

## 2. 지원 대상
### 위기상황 인정 사유
- 주소득자가 사망, 가출, 행방불명 등으로 소득을 상실한 경우
- 중한 질병 또는 부상을 당한 경우
- 가구구성원으로부터 방임, 유기, 학대 등을 당한 경우

## 3. 소득 및 재산 기준
- 소득: 기준 중위소득 75% 이하 (단, 1인 가구는 120% 이하)
- 재산: 대도시 2억 4,100만원, 중소도시 1억 5,200만원, 농어촌 1억 3,000만원 이하
- 금융재산: 600만원 이하 (주거지원은 800만원 이하)

## 4. 지원 내용
- 생계지원: 4인 기준 월 1,833,500원
- 의료지원: 300만원 이내
- 주거지원: 대도시 66만원 이내
